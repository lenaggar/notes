# whoami
`whoami` prints the username of the currently active user (logged in user).

# hostname
`hostname` prints the name of the computer the user is connected to (useful when managing multiple servers).

# pwd
`pwd` **p**rints the **w**orking **d**irectory, this is where the user is at currently in the file system.

# ls
`ls` prints a list of the files and folders in the current directory.

# date
`date` prints the current date and time according to the system.

# clear
`clear` clears all text on the shell screen to give a fresh screen (`ctrl+L` is a shortcut).

# echo
`echo` prints whatever it is given. It is a useful command when it comes to scripting and moving text between scripts or injecting some text into a script especially when paired with the pipe `|` for redirection.

# cat
`cat` prints the content of a file to the shell output without opening the file in a text editor application.

# uptime
`uptime` prints how long the system has been up and running for.

# touch
`touch` creates a file with name provided by the user as first argument in the current directory.

# rm
`rm` deletes a file.

# history
`history` prints a list of the commands the user entered to the shell.
