# Main commands of a linux package manager
- `sudo apt update` update repositories
- `sudo apt upgrade` upgrade the packages on the system
- `apt search htop` search for a package
- `apt show htop` display information about a package
- `sudo apt install htop` install a package
- `sudo apt remove htop` uninstalls a package
- `sudo apt purge htop` uninstall everything related to a package including configs
- `apt list --installed | grep htop` lists installed packages

# Bonus
- `tree` creates a text output representation tree of the file structure of a given directory
