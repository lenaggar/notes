- `whoami` => prints the active use name 
- `who` => prints who is logged on
- `w` => prints who is logged on and what they are doing
- `id` => prints the user id `uid`, the user's main group id `gid` (because each user has their own group), and other `groups` that the user is part of
	- `id` => `uid=1000(mahmoud) gid=1000(mahmoud) groups=1000(mahmoud),966(docker),984(video),990(lp),993(input),998(wheel)`
- `groups` => prints the groups a user is part of
	- `groups` => `mahmoud docker video lp input wheel`
- Each user has a `username` (like `mahmoud`), a `userId` (like `1000`), a `homeDir` (like `/home/mahmoud`), and a `defaultShell` (like `/usr/bin/bash`). This information is stored in `/etc/passwd` in the format: `usename:password:uid:gid:comment:home:shell`
	- `cat /etc/passwd` => contains a line for the current user like:
	  `mahmoud:x:1000:1000::/home/mahmoud:/usr/bin/bash`
	- the `x` in the previous command means that the password is stored elsewhere, `/etc/shadow`
	- all the groups on a system are in `/etc/group`

## permissions
- `-rw-r--r-- 1 root root size MMM dd hh:mm file_name`
	- the first byte `-` indicated the type of the file (`-` for file, `d` for directory, `l` for symbolic link, `c` for character device, `b` for block device, `s` for socket, `p` for named pipe)
