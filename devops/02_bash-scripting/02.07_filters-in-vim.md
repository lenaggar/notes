Create `~/.local/bin` directory (following the XDG Base Directory convention) and make sure it exists in the `PATH` variable

```bash
mkdir -p ~/.local/bin
echo 'export PATH="$HOME/.local/bin:$PATH"' >> ~/.bashrc
source ~/.bashrc
```

### upper
```bash
#!/bin/bash
read -r line
echo "${line^^}"
```

### lower
```bash
#!/bin/bash
read -r line
echo "${line,,}"
```

### gendate (Insert Today’s Date)
```bash
#!/bin/bash
date +%Y-%m-%d
```

### slugify
```bash
#!/bin/bash
read -r line
line="${line,,}"
line="${line//[^a-z0-9]/-}"  # ^ means negated because it's in brackets
line="${line//--/-}"
echo "${line#-}"
```
`echo "Hello World! This is a Test" | slugify` => `hello-world-this-is-a-test`
